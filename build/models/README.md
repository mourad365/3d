# Models Directory / مجلد النماذج

Place your `.glb` 3D model files in this directory.

ضع ملفات النماذج ثلاثية الأبعاد بصيغة `.glb` في هذا المجلد.

## Supported Format / الصيغة المدعومة

- GLB (Binary glTF)

## Usage / الاستخدام

1. Add your `.glb` files to this folder
2. The viewer will automatically detect and list them
